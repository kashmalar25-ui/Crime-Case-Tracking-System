# Crime Case Tracking & Investigation Report System

## Overview

The Crime Case Tracking & Investigation Report System is a Java-based application developed as part of the Object-Oriented Programming course. The system is designed to manage criminal cases, investigation records, and report generation efficiently.

---

## Features

- Register new crime cases
- Store investigation details
- Manage criminal records
- Generate investigation reports
- Search and update case information

---

## Technologies Used

- Object-Oriented Programming
